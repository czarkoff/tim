tim by czarkoff

Tiny image manipulator

The idea of a program came to me when I had to create 4:3 thumbs for a group of different orientation images. I had to do my cropping with GIMP which isn't so suitable for such kind of manipulations.

The program takes an image and applies a given set of filters. One of them - cropping - is done in graphical mode.

Dependencies

  • python 2.x (http://www.python.org/)
  • Tkinter (included with python)
  • PIL (http://www.pythonware.com/products/pil/)
  • Any graphical UI capable of Tk (X11, Windows, MacOS X etc.)

Install

The program conists of a single executable python script, so on UNIX-like You only need to make "tim" script executable and put it somewhere within $PATH, eg.: 

$ chmod +x tim
$ cp tim ~/bin/tim

On Windows You should probably place a script to some known place and invoke it as "python /path/to/tim infile op1 arg op2 arg outfile" (given that Your python installation is done the wright way.

Documentation

Currently the project contains a single useful file that can be chmodded and placed wherever one wants, so I don't think adding a manpage is a great idea. 

 $ tim -h
usage: tim infile [-c GEOMERTY] [-r ANGLE] [-s GEOMETRY] outfile

options (filters):
  -c GEOMETRY  crop an image according to a GEOMETRY spec specified as:
               "WIDTHxHEIGHT[+LEFT+TOP]"; if LEFT and TOP are ommited,
               the UI is brought up to help You set them.
  -r ANGLE     rotate an image ANGLE degrees clockwise (if positive value)
               or counter clockwise (if negative)
  -s GEOMETRY  scale an image according to a GEOMETRY spec specified as:
               [!]WIDTHxHEIGHT or PERCENT%.
               By default, the image is resized to fit a box with the
               GEOMETRY specification while maintaining the aspect ratio.
               "!" in the beginning of the string overrides this behaviour
  -h           print this help

Filters are applied in order specified, eg.:

    tim old.jpg -s !500x700 -r 90 -c 100x100+20+20 -s 500x700 new.png

will import JPEG image, scale it to 500x700 regardless initial aspect ratio,
rotate the scaled image 90 degrees clockwise, crop rectangular area with
coodinates (20,20)-(120,120) out of the rotated and scaled image scale it
again to fit 500x700 box maintaining aspect ration and finaly export
the resulting image to file new.png in PNG format.

Running tim without filters is still useful as a mean of format conversion.
